## API

- label `boolean`: 标签的显示模式
- change `boolean`: 左右变换
- list `array`: ITEM 对象

### ITEM

- img `string`: 图片链接
- title `string`: 标题
- label `string`: 标签
- cnt `string`: 段落内容
- change `boolean`: 左右顺序(false 为正序)
