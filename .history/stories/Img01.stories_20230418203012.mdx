## API

- id `boolean`: 是否显示序号
- list `array`: ITEM 对象

### ITEM

`cnt` 和 `list` 只有一个可以显示。

- img `string`: 图片链接
- title `string`: 一级标签
- label `string`: 二级标签
- cnt `string`: 段落内容
- list `array`: 内容数组
