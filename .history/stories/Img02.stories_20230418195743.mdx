## API

- btn `boolean`: 是否显示按钮
- list `array`: ITEM 对象
- list 中的 ITEM 对象最多三个

### ITEM

- img `string`: 图片链接
- title `string`: 标题
- cnt `string`: 段落内容
- btn `string`: 按钮文本
- link `string`: 按钮跳转链接
