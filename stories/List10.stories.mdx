## API

- list `array`: ITEM 对象
- link `boolean`: 是否显示跳转标签
- lbLink `boolean`: 跳转标签的显示模式

### ITEM

- icon `string`: 图标链接
- title `array`：标题数组
- cnt `string`：段落内容
- src `string`：跳转链接
- link `string`：跳转链接说明
