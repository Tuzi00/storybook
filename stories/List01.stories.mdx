## API

- id1 `boolean`: 是否显示ITEM对象内title序号
- id2 `boolean`: 是否显示list序号
- id3 `boolean`: 是否显示ITEM对象内list序号
- list `array`: ITEM 对象
- clr `string`: 背景颜色

### ITEM

- title `string`: 一级标签
- cnt `string`: 段落内容
- list `array`: 内容数组
