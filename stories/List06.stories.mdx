## API

- list `array`: ITEM 对象

### ITEM

- name `string`: 一级标签
- subtitle `string`: 一级副标签
- desc `string`: 小字内容
- rec `array`: 内容数组
- list `array`: 二级ITEM 对象

#### 二级ITEM

- m `string`: 内容
- v `string`: 内容
- d `string`: 内容
- o `string`: 内容
